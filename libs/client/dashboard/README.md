# dashboard

This library was generated with [Nx](https://nx.dev).

## Overview

The dashboard library provides a customizable trading desk dashboard with dynamic widget rendering based on feature flag configuration.

## Features

- Dynamic widget layout controlled by FeatureService
- Market Depth widget for L2 order book visualization
- Risk Matrix widget for portfolio VaR display
- News Terminal widget for financial news feed
- Responsive grid layout
- Signal-based reactive state management

## Running unit tests

Run `nx test dashboard` to execute the unit tests.
